# handshake

# Env

- Node.js: 14.2.0
- react-native-cli: 2.0.1
- react-native: 0.62.2

# Get Started

```sh
% git clone https://github.com/kaw-k/handshake.git
% cd handshake
% npm i

# for iOS
% cd ios
% pod install
```
